About the app

Monect PC Remote is a free app that allows you to control PC over Wi-Fi or Bluetooth locally or remotely.


Key Features:
- Play all kinds of PC games with on board sensors and specifically designed button layouts and you can edit them as you wish
- Transfer screen and camera of PC to phone in real time and vice verse
- File transfer between devices
- Highest security: 256 Bit AES Session Encoding on remote network


How to use:
Please install PC Remote Receiver on your computer before using PC remote.
Download the PC Remote Receiver at our official site for free:
